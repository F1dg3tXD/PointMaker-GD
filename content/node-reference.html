🧑‍💻 Node Reference

PointMaker GD offers several specialized nodes to implement various interactive behaviors. Click on a node in the Table of Contents to the left to view its detailed documentation, including properties, signals, and usage examples.

The core nodes are:

  • PointTrigger: For basic interactions like clicks or area enters.
  • PointHover: Detects mouse hover events.
  • PointHold: For interactions requiring a sustained mouse press.
  • PointDrag: Makes objects draggable.
  • PointSnap: Allows draggable objects to snap into designated areas.
  • PointRadial: Creates rotatable knob-like controls.
  • PointSliderH / PointSliderV: Implements horizontal and vertical sliders along a path.
