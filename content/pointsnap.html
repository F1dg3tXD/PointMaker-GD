PointSnap

Description:
Allows a PointDrag node to snap into place, with optional alignment, feedback, and locking.

Properties:

  • accepted_dragger: NodePath — Only accept this drag node (if allow_any_dragger is false).
  • allow_any_dragger: bool — Accept any drag node.
  • require_specific_dragger: bool — Only accept the specific dragger if true (used with allow_any_dragger).
  • animation_player_path: NodePath — Path to an AnimationPlayer node.
  • snap_in_animation: String — Animation to play when snapped in.
  • snap_out_animation: String — Animation to play when unsnapped.
  • correct_snap_sound: AudioStream — Sound to play on correct snap.
  • wrong_snap_sound: AudioStream — Sound to play on wrong snap.
  • unsnap_sound: AudioStream — Sound to play when unsnapped.
  • align_rotation: bool — Align rotation when snapped.
  • align_scale: bool — Align scale when snapped.
  • lock_when_snapped: bool — Lock the drag node when snapped.

Signals:

  • snapped(dragger: Node) — Emitted when a dragger is successfully snapped.
  • unsnapped(dragger: Node) — Emitted when a snapped dragger is released.
  • snap_failed(dragger: Node) — Emitted if snapping fails (e.g., already snapped).
  • rejected_dragger(dragger: Node) — Emitted if a dragger is not accepted for snapping.

Usage:

  • Add as a child of your scene.
  • Add a CollisionShape2D or CollisionPolygon2D as a child.
  • Configure properties in the Inspector.
  • Place in the scene where you want objects to snap.
