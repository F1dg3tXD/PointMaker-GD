PointSliderH / PointSliderV

Description:
Horizontal (PointSliderH) and vertical (PointSliderV) sliders along a Path2D.

Properties:

  • handle_path: NodePath — Path to the handle sprite/node to move.
  • path2d_path: NodePath — Path to the Path2D node.
  • value: float — Initial value (0.0–1.0).
  • step: float — Snap increment (0 for smooth).

Signals:

  • value_changed(value: float) — Emitted when the value changes during drag.
  • value_set(value: float) — Emitted when the value is set (on release).

Usage:

  • Add as a child of your scene.
  • Add a CollisionShape2D or CollisionPolygon2D as a child.
  • Configure properties in the Inspector.
  • Set handle_path and path2d_path to the appropriate nodes.
