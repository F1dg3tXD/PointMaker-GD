PointHold

Description:
Triggers after holding the mouse button over the area for a set duration.

Properties:

  • required_hold_time: float — Seconds to hold before triggering.
  • hold_loop: bool — If true, animation loops while held.
  • cancel_on_move: bool — Cancels hold if mouse leaves area.
  • animation_player_path: NodePath — Path to an AnimationPlayer node.
  • animation_name: String — Name of the animation to play.

Signals:

  • hold_started — Emitted when holding starts.
  • hold_triggered — Emitted when hold time is reached.
  • hold_released — Emitted when the hold is released or canceled.

Usage:

  • Add as a child of your scene.
  • Add a CollisionShape2D or CollisionPolygon2D as a child.
  • Configure properties in the Inspector.
