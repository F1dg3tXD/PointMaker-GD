PointRadial

Description:
A rotatable knob for value selection, e.g., dials or rotary controls.

Properties:

  • knob_path: NodePath — Path to the knob sprite/node to rotate.
  • min_angle: float — Minimum angle (degrees).
  • max_angle: float — Maximum angle (degrees).
  • use_limits: bool — If true, restricts rotation to min/max.
  • start_value: float — Initial value (0.0–1.0).
  • step: float — Snap increment (0 for smooth).

Signals:

  • value_changed(value: float) — Emitted when the value changes during drag.
  • value_set(value: float) — Emitted when the value is set (on release).

Usage:

  • Add as a child of your scene.
  • Add a CollisionShape2D or CollisionPolygon2D as a child.
  • Configure properties in the Inspector.
  • Set knob_path to the node you want to rotate.
